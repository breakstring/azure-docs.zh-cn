# <a name="partner-ecosystem-for-azure-api-for-fhir"></a>Azure API for FHIR 的合作伙伴生态系统
Azure API for FHIR 已向所有 Azure 客户发行 Azure API for FHIR 正式版，对此我们感到非常兴奋。 对于大家可以使用本公司的服务构建的解决方案，我们尤其感到振奋。
