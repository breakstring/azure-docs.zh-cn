# <a name="pi-azure-cosmos-db"></a>PI (Azure Cosmos DB)
[!INCLUDE[appliesto-sql-api](includes/appliesto-sql-api.md)]

 返回 PI 的常量值。  
  
## <a name="syntax"></a>语法
  
```sql
PI ()  
```  
   
## <a name="return-types"></a>返回类型
  
  返回数值表达式。  
  
## <a name="examples"></a>示例
  
  下面的示例返回 `PI` 的值。  
  
```sql
SELECT PI() AS pi 
```  
  
 下面是结果集：  
  
```json
[{"pi": 3.1415926535897931}]  
```
