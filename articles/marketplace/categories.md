# <a name="categories-and-subcategories-in-the-commercial-marketplace"></a>商业应用商店中的类别和子类别

类别和子类别基于解决方案类型映射到一个或两个 [商业 marketplace 在线商店](overview.md#commercial-marketplace-online-stores) 。 当你在合作伙伴中心创建商业市场产品/服务时，它可能会根据产品/服务类型、类别和你选择的其他选项在任一在线商店或同时在两个在线商店列出。 在极少数情况下，产品/服务与两个在线商店的用户都相关，它便可以同时在两个在线商店中显示。

选择最适合你的解决方案的类别和子类别。 分配适用于你的产品/服务的类别和子类别可帮助客户在你的商业应用商店中找到你的产品/服务。 

可以选择：

- 至少一个和多达两个类别。 您可以选择主要和次要类别。
- 每个主类别最多两个子类别。 如果未选择任何子类别，你的产品/服务仍会在所选类别下发现。
