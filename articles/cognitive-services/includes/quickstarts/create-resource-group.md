## <a name="create-a-resource-group"></a>创建资源组

在创建认知服务资源之前，帐户必须具有 Azure 资源组才能包含该资源。 如果还没有资源组，请在 [Azure 门户](https://ms.portal.azure.com/) 中创建一个资源组，然后再继续。
