若要使用最新版本的文本分析 v3 客户端库，你将需要下载命名空间中的最新软件包 `Azure.AI.TextAnalytics` 。 [快速入门文章](../quickstarts/client-libraries-rest-api.md)中的 "**设置**" 部分列出了可用于首选语言的命令，其中包含示例代码。
