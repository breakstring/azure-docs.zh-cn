此实体类别包括财务信息和官方形式的标识。 从模型版本开始可用 `2019-10-01` 。 下面列出了子类型。 

### <a name="financial-account-identification"></a>财务帐户标识

| 子类型名称               | 说明                                                                |
|----------------------------|----------------------------------------------------------------------------|
| ABA 路由号码        | 美国银行家协会 (ABA) 传输路由号。                  |
| SWIFT 代码                 | 付款说明信息的 SWIFT 代码。                           |
| 信用卡                | 信用卡号。                                                       |
| 国际银行帐号 (IBAN)                  | 付款说明信息的 IBAN 编码。                            |
