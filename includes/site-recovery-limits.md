以下限制适用于 Azure Site Recovery。

| 限制标识符 | 限制 |
| --- | --- |
| 每个订阅的保管库数 |500 |     
| 每个恢复服务保管库的服务器数目 |250 |
| 每个恢复服务保管库的保护组数目 |无限制 |
| 每个恢复服务保管库的恢复组数目 |无限制 |
| 每个保护组的服务器数 |无限制 |
| 每个恢复计划的服务器数 |100 |
