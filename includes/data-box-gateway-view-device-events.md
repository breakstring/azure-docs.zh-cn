在 Azure 门户中执行以下步骤以查看设备事件。 

1. 在 Azure 门户中，请切换到资源，然后继续 **监视 > 设备事件**。
2. 选择一个事件，然后查看警报详细信息。 采取相应措施，消除警报条件。

    ![选择事件并查看详细信息](media/data-box-edge-gateway-view-device-events/view-device-events.png)
