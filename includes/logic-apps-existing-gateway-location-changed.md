**问**：我的现有网关的位置为何发生了更改？ <br/>
**答**：2017 年 5 月 3 日之前创建的网关资源已经移动到了创建这些网关的工作或学校帐户的 Azure AD 租户的原始位置。 但是，这些更改应当不会影响当前正在运行的逻辑应用，这些应用应当继续照常运行。 随着该网关在五月份的正式发布，网关资源的位置可能不同于逻辑应用位置。
